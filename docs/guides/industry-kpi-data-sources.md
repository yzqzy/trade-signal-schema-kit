# 行业 KPI 与公开数据源边界

本项目的 Topic 研报采用“通用主链路 + 行业 Profile”的数据策略：通用字段优先从年报、公告、交易所/巨潮、企业行动、财务三表取得；行业 KPI 只在匹配行业 profile 后启用，不作为所有公司的必填项。

## 当前数据可稳定支撑

| 数据域 | 主来源 | 当前用途 | 备注 |
| --- | --- | --- | --- |
| 年报与公告 | 巨潮、交易所公告、公司 IR | 年报 PDF、分红政策、诉讼担保、关联交易、客户数据公告 | 法定披露优先级最高 |
| 企业行动 | Feed corporate actions | DPS、除权除息、分红连续性 | 可支撑 DDM 与股东回报 |
| 财务三表 | Feed financial snapshot/history | 收入、利润、OCF、Capex、资产负债、费用率、营运资本 | 可支撑穿透回报与估值 |
| 历史估值 | Feed historical PE | PE Band、历史分位 | 用于估值区间 |
| 监管事件 | 交易所/巨潮监管事件聚合 | 问询、监管措施、纪律处分 | 官方源无命中只写“未形成确认事件” |
| 同业池 | Feed peers TopN | 财务规模与盈利对标 | 不固定同行名单 |
| 分红政策文字 | 年报/公告/F10 经营画像 | `shareholderReturns` 信号 | 已在 Phase1A 做通用增强 |

## 年报结构化抽取边界

Phase2A/Phase2B 已把年报关键块扩展为两组：

| 组别 | Section | 用途 |
| --- | --- | --- |
| Turtle 财务风险块 | `P2` / `P3` / `P4` / `P6` / `P13` / `MDA` / `SUB` | 受限资产、应收账款、关联交易、或有事项、非经常性损益、MD&A、子公司 |
| 经营画像块 | `BUSINESS` / `SEGMENT` / `OPERATING` / `CAPEX` / `DIVIDEND` | 主营业务、分部收入、行业专属经营指标、资本开支、分红政策 |

其中 `BUSINESS`、`SEGMENT`、`CAPEX`、`DIVIDEND` 适合作为多数行业的通用证据块；`OPERATING` 只负责定位经营指标文本，字段级解析必须交给行业 profile，避免把电信 ARPU、乳业渠道、银行 NIM 等不同口径混成一套通用字段。

## 行业 Profile 建议

| Profile | 典型行业 | 建议 KPI | 免费公开来源 | 接口补充优先级 |
| --- | --- | --- | --- | --- |
| `telecom` | 通信运营商 | 移动客户、5G 客户、宽带客户、ARPU、DICT/政企收入、算力/云收入、Capex | 公司 IR 客户数据页、客户数据公告、年报、工信部通信业统计 | 高 |
| `dairy_food` | 乳业/食品饮料 | 产品分部收入、渠道/区域、经销商、库存、原奶价格、食品安全事件 | 年报、公告、国家统计局/行业协会、市场监管抽检 | 中 |
| `bank` | 银行 | NIM、不良率、拨备覆盖率、贷款结构、存款成本 | 年报/季报、监管披露 | 中 |
| `insurance` | 保险 | NBV、保费结构、综合成本率、偿付能力、投资收益率 | 年报、偿付能力报告、公司公告 | 中 |
| `manufacturing` | 制造/新能源 | 产能、出货、订单、单价、库存、毛利、Capex 分项 | 年报、公告、互动易/业绩说明会 | 中 |
| `real_estate` | 地产 | 销售额、土储、竣工交付、融资成本、三道红线、预收款 | 年报、公告、交易所问询、行业统计 | 中 |

## 不建议作为默认主链路

- WebSearch/新闻：只作为开放背景补充，不作为财务、估值、监管核心证据。
- 市场监管、食品抽检、司法执行：只在行业或文本触发时查询，例如食品、医药、地产、消费服务等。
- 付费行业数据库：可做增强源，但不能成为开源/免费默认链路。

## 接口补充建议

最值得补的是“年报/公告专项抽取”而不是泛化 WebSearch：

1. 公告分类检索：利润分配、权益分派、回购、客户数据、经营数据。
2. 年报章节结构化：主营构成、分部收入、经营指标、资本开支、分红政策。
3. 行业 profile adapter：按行业启用 KPI 规则，例如 `telecom` 解析客户数据公告，`dairy_food` 解析产品/渠道/区域表。
4. 专源触发规则：只有命中行业或风险词时，才查市监、食品抽检、司法执行等专源。

## 免费公开平台方向

| 平台 | 适合数据 | 说明 |
| --- | --- | --- |
| 巨潮资讯 | 年报、公告、分红、回购、问询回复、经营数据公告 | A 股法定披露主链路 |
| 上交所/深交所/北交所 | 公告、监管问询、纪律处分、监管措施 | 与巨潮互补，监管事件优先级高 |
| 国家统计局国家数据 | 宏观、行业、区域、年度/月度统计 | 适合行业背景，不适合公司专属 KPI |
| 工信部/地方通信管理局 | 通信业收入、用户、5G、千兆、物联网、流量 | `telecom` profile 的行业背景源 |
| 中国上市公司协会 | 上市公司行业统计分类 | 适合行业映射和 profile 选择 |
| 行业主管部委/协会 | 行业产量、价格、监管政策 | 需要按行业逐个 adapter 化 |
